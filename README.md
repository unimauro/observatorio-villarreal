# Observatorio Villarreal

Portal ciudadano de transparencia de la **Universidad Nacional Federico Villarreal** (UNFV, pliego 524). Solo datos públicos.

**Live:** https://unimauro.github.io/observatorio-villarreal/

Módulos: presupuesto y ejecución (MEF/SIAF), en qué se gasta cada sol, proveedores y dueños (OECE/CONOSCE), investigación (OpenAlex), gasto↔valor, personal (Portal de Transparencia). Nunca nombres de estudiantes.

Red de observatorios: [UNI](https://unimauro.github.io/observatorio-uni/) · [San Marcos](https://unimauro.github.io/observatorio-sanmarcos/) · [UNSAAC](https://unimauro.github.io/observatorio-unsaac/) · [ENSAD](https://unimauro.github.io/observatorio-ensad/). No es un portal oficial. Correcciones: carlos@cardenas.pe
